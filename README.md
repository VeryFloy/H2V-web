# H2V Messenger — Web Frontend

Frontend-only repository for [H2V Messenger](https://web.h2von.com).

## Quick Start

```bash
git clone https://github.com/VeryFloy/H2V-web.git
cd H2V-web/frontend
npm install
npm run dev
```

Open **http://localhost:5173**.

See [frontend/README.md](frontend/README.md) for full documentation.
